AUTHENTICATION
================================================================================

Login Flow:
1. User enters email and password on login page
2. Backend validates credentials
3. Backend returns JWT token
4. Token stored in browser localStorage
5. Token sent with all API requests automatically
6. User can logout to clear token

================================================================================
TROUBLESHOOTING
================================================================================

Issue: "Cannot connect to backend"
Solution:
  - Ensure backend is running on configured port
  - Check .env has correct REACT_APP_API_URL
  - Verify backend CORS settings

Issue: "Login fails with invalid credentials"
Solution:
  - Verify admin exists in backend database
  - Check backend is responding to login endpoint
  - Look at browser Network tab to see actual error

Issue: "Styles not showing correctly"
Solution:
  - Restart development server: npm start
  - Clear browser cache (Ctrl+Shift+Delete)
  - Run: npm cache clean --force

Issue: "API calls failing"
Solution:
  - Open browser Developer Tools (F12)
  - Check Network tab for failed requests
  - Look at Response tab for error details
  - Verify backend endpoints match expectations
